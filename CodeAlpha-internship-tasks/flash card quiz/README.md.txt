An interactive flashcard quiz app built using HTML, CSS, and JavaScript.  
Users can flip cards to reveal answers, move between cards, and enjoy smooth animations.

---
     Description
This project helps users study using digital flashcards.  
Each card displays a question on the front and flips to show the answer.  
Navigation buttons allow switching between cards smoothly.

---

     Features
- Flip animation for cards  
- Next / Previous card navigation  
- Clean modern UI  
- Fully responsive  
- Vanilla JavaScript (No libraries)

- JavaScript
